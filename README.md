# AI-Candidate-Evaluation-and-Scoring-System

## Description:
Developed an automated scoring workflow that processes form submissions, categorizes users based on gender and profession, evaluates responses using AI, and updates records dynamically. The system demonstrates workflow automation, conditional logic, and AI-driven decision-making using Google Gemini and database integration.
## Project Overview
The workflow receives user data from a form submission and automatically routes the data through different branches based on gender and profession. The AI agent analyzes the input and generates a profession-based score. Finally, the system updates the database record with the calculated result.

## Workflow Nodes Used

### On Form Submission
Triggers the workflow when a user submits a form.

### Male/Female (Condition Node)
Checks the gender field and routes the workflow accordingly.

### Male
Processes records for male users.

### Female
Processes records for female users.

### Rating Based on Profession
Determines scoring logic based on the selected profession.

### Google Gemini Chat Model
Provides the AI model used for generating scores and analysis.

### Update Record
Updates the database with the calculated score.

## Workflow Process

1. User submits a form
2. Workflow is triggered automatically
3. System checks gender (Male or Female)
4. Workflow routes data based on profession
5. AI evaluates the user input
6. Score is calculated based on profession
7. Database record is updated with results
